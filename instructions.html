How to Install USM-eM

USM-eM for Nintendo 3DS Installation Instructions:

Prerequisites:

- A Nintendo 3DS family console running Luma 8.0 or higher.

Info on Luma installation from normal consoles without any modifications, this can be found here. (Note that this guide assumes you have followed these instructions and have Luma3DS insatlled).
If you are running Luma 7.0.5 or below, update to boot9strap here.
If you are running Luma 7.1, AuReiNAND, or any other CFW, update to boot9strap here.

- A copy of Super Smash Bros. for Nintendo 3DS
- A device capable of transferring files to an SD card.

Instructions:

- Download ModMoon. Be sure to read its instruction manual, it's a really powerful tool! The 3dsx version can be run from the Homebrew Launcher, while the CIA version can be installed to the HOME menu. This guide covers the CIA version.
- Download the latest version of USM-eM from our releases page. - On the root of your 3DS' SD card, enter the "3ds" folder. Create it if it doesn't exist.
- In the "3ds" folder, drop the ModMoon download you grabbed earlier.
- Create a "ModMoon" folder within the "3ds" folder.
- Depending on the region of your game, create a folder called "00040000000EDF00" if you have a version of Smash from the USA, "00040000000EE000" if it's a European version, and "00040000000B8B00" if it's a Japanese version.
- Extract the root of the USM-eM mod to the folder you just created.
- Turn on your 3DS while holding select to access the Luma configuration menu.
- Turn on "Enable game patching". This is required to load mods properly.
- Open ModMoon. If you downloaded the 3dsx version, simply open the Homebrew Launcher and you should find it, and you can skip these next steps. If you grabbed the CIA version, do this:
- Open FBI and select "SD".
- Navigate to the "3ds" folder.
- Select the ModMoon CIA file, and install it.
- Return to the HOME menu, and open the newly installed ModMoon.
- Follow the on-screen tutorial, being sure to select Smash in your active titles menu and setting it as your current title.
- Make sure that a modpack with a description of either "USM-eM - Release 0.1" or "Winds" is selected, then launch Smash!
- The title screen should now show the USM-eM logo rather than the default logo, this means the mod loaded properly!
- The selected Smash modpacks will always play after Smash is launched from the HOME Menu whether you launch Mod-Moon beforehand or not. If you want to launch vanilla Smash, launch Mod-Moon and disable the active title. Enable it to re-enable loading modpacks.

USM-eM for Nintendo Wii U Installation Instructions:

Prerequisites:

- A Wii U on version 5.5.1 or 5.5.2
- A copy of Super Smash Bros. for Nintendo Wii U
- A way to access the Homebrew Launcher
- SDCafiine 1.4 installed

Instructions:

- Download and open the latest Wii U version from the Releases page.
- Copy the folder called USM-eM_v0.1 to your Smash mod folder.
         SD:\sdcafiine\0005000010145000\USM-eM_v0.1 for EU
         SD:\sdcafiine\0005000010144F00\USM-eM_v0.1 for US
- Insert your SD card into your Wii U.
- Load the Homebrew Launcher through any method of your choice, then load SDCafiine.
- You should be sent back to the HOME Menu, so now open Smash.
- If you have other modpacks in your Smash mod folder, select USM-eM_v0.1. If you don't have any other modpacks, skip this step.
- Once Smash has loaded completely, the USM-eM logo should appear in place of the Smash 4 logo - this means the mod has loaded successfully!

*Note that the mod will continue to run in place of Smash 4 as long as the console is turned on. Once it’s turned off, it’s cleared from memory, and until you load SDCafiine again, Smash 4 will load normally - the mod isn’t persistent.

USM-eM is a free, non-profit mod of Super Smash Bros. for Wii U and 3DS.
The USM-eM team does not support any form of piracy; the mod requires all users to own a copy of the game for either system, and no paid DLC will be unlocked by using this mod.
